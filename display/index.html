Display

Definición

Pantalla, visualización, muestra. Lo que se muestra en la pantalla

La propieddad "display" especifica si un elemento es tratado como block o inline element y el diseño de lxs hijxs.


Display None

Oculta el elemento completamente a la vista, haciéndolo invisible. Permite dejar de visualizar contenido dentro del sitio.

Cuando ocupamos uno de estos, las etiquetas dejan de verse pero el código sigue ahí.

Display Block

Muestra el elemento como un elemento a nivel de bloque.

Se utiliza para mostrar el elemento "displayBlock" Ocupa todo el ancho disponible y comienza en una nueva línea. Es útil para crear secciones distintas en tu página web, como encabezados, párrafos o elementos de formulario

Display Inline

Muestra el elemento en línea con el texto circundante Adecuado para elementos de texto pequeños, imágenes o enlaces en línea. .

Trata el elemento como si fuera parte del flujo de texto.

Significa que el contenedor o bloque se adapta al ancho de su contenido hijx.

Si le pones tamaño, ignora las propiedades de alto (height) y ancho (width).

El contenido solo ocupa el espacio que necesita.
